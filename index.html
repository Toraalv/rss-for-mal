MyAnimeList airing anime RSS feed & Waybar module

This is a simple API wrapper for MyAnimeList that fetches a requested users currently watching, and airing, anime, and then fetches the broadcast schedule for these. The data is then presented as either RSS or as a JSON formatted string, specifically made for Waybar.
Feedback or suggestions are welcome, reach me at rfm@toralv.dev

Waybar module:

		
// change format to whatever you want
// the values returned by the API is "text" and "tooltip"
// text is the amount of animes that airs today
// tooltip is the schedule for all anime that you're watching and is currently airing

// change interval to whatever fits you best
// see possible values at: https://github.com/Alexays/Waybar/wiki/Module:-Custom

// remember to put "custom/rfm" in either modules-left, modules-center or modules-right

"custom/rfm": {
	"format": "MAL {}",
	"return-type": "json",
	"interval": "3600",
	"exec": "curl -k https://rfm.toralv.dev/waybar/$USER"
}
example output that waybar then reads: 
		
$: curl -k https://rfm.toralv.dev/waybar/Toralv
